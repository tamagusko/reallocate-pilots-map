# REALLOCATE Pilots Map

This Streamlit application visualises the geographic areas of REALLOCATE project pilots.

It displays pilot areas on an interactive map using Folium and allows users to:

- View all pilots together
- Zoom in on individual pilots
- See pilot/city info on click
- Download the corresponding GeoJSON files

## Usage

1. Place your `.geojson` files in the `data/` folder.
2. Run the app:

```bash
streamlit run streamlit_app.py
````
